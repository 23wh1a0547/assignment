# 🌟 NayePankh Foundation - Front-End Internship Project

This is a responsive and modern website developed for the **NayePankh Foundation Front-End Development Internship Task** using only **HTML and CSS**. The project represents an NGO platform showcasing its initiatives, internships, certificates, and volunteer registration system.

---

## 🎯 Objective
To create a clean, responsive, and visually appealing NGO website that highlights social initiatives, internship opportunities, and community engagement activities.

---

## 🚀 Features

- Sticky Navigation Bar with smooth scrolling  
- Hero Section with background image and animation  
- Multiple Sections: About, Initiatives, Internships, Certificates, Contact  
- 🌙 Dark Mode using CSS-only checkbox toggle  
- 🎨 Modern card-based UI design  
- ✨ Hover effects and smooth transitions  
- 📱 Fully responsive design (mobile + tablet + desktop)  
- 🖼️ Certificate gallery with images  

---

## 🛠️ Technologies Used
HTML5, CSS3, CSS Grid, Flexbox, Media Queries, CSS Animations
